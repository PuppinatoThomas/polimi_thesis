1. **Standardize Missing Values:** Replace all placeholders (`-1`, `'Unknown'`, `'-'`) with proper NaN
2. **Type Conversion:**

   - Remove dollar signs from prices
   - Standardize date formats
3. **Outlier Treatment:** 
   - Remove or flag negative values in numeric fields
   - Investigate extremely small house sizes
4. **Data Validation:**
   - Validate state abbreviations vs full names
   - Standardize address formatting
   - Verify zip code validity

This dataset requires significant cleaning before reliable analysis, with particular attention to the placeholder values and type inconsistencies that affect most numeric columns.
